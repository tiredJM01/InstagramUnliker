# instagram-unlike-userscript
Automate unliking instagram posts. 

Cooldown and batch size is configurable.
I vibe coded this using Claude cause some stupid extension decided to follow 4.4k random posts and it made me look like a creep.

Do what you want but try not making cooldown to fast or u might get flagged for botting or something.


Use some userscript manager like tampermonkey to run it.

TO USE:

GO TO https://www.instagram.com/your_activity/interactions/likes/ and click the start button at the bottom right.
